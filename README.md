# ChatRoom

*基于JSP开发的简易聊天室*

>JavaEE编程技术

## 功能描述

* 普通用户登录成功后进入聊天室，聊天室可显示以往聊天记录，可发言，刷新聊天记录

* 登录页面，用户名输入框显示上一次成功登录的用户名，快捷登录

* 主聊天室页面 可显示 当前在线人数 及 在线用户名称

* 普通用户登录，实现用户的登录、注册、修改密码及删除功能（增删改查）

* 管理员登录，实现用户录入、删除、修改、多条件查询并显示功能

* 实现简单的三层架构
